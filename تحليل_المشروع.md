# تحليل شامل لمشروع نظام نقاط البيع (POS)

## 📋 نظرة عامة

هذا مشروع **نظام نقاط البيع (Point of Sale)** مطور باستخدام **Flutter** للمطاعم. المشروع يدعم اللغة العربية بشكل كامل ويحتوي على جميع الوظائف الأساسية لنظام POS متكامل.

---

## 🏗️ البنية التقنية

### التقنيات المستخدمة

- **Framework**: Flutter (SDK ^3.9.2)
- **قاعدة البيانات**: SQLite (sqflite)
- **State Management**: Provider
- **الطباعة**: دعم طابعات ESC/POS الحرارية
- **الرسوم البيانية**: Syncfusion Flutter Charts, fl_chart
- **معالجة الصور**: image_picker, image package
- **الخطوط**: Tajawal (خط عربي)

### البنية المعمارية

```
lib/
├── main.dart (8325 سطر - ملف واحد يحتوي على كل الكود)
└── old/ (ملفات قديمة محفوظة)
```

**ملاحظة مهمة**: جميع الكود موجود في ملف واحد (`main.dart`) مما يجعل الصيانة صعبة.

---

## 🗄️ قاعدة البيانات

### الجداول الرئيسية

1. **users** - المستخدمين والموظفين
   - id, name, pin, role

2. **categories** - فئات المنتجات
   - id, name, printerName, printerId

3. **products** - المنتجات
   - id, name, categoryId, costPrice, sellPrice, laborCost
   - imagePath, stock
   - priceSmall, priceMedium, priceLarge (للمقاسات)

4. **addons** - الإضافات على المنتجات
   - id, productId, name, price, categoryId

5. **discounts** - العروض والخصومات
   - id, name, type (PERCENT/FIXED), value
   - startDate, endDate
   - targetCategoryId, targetProductId

6. **sales** - المبيعات
   - id, invoiceId, totalAmount, date
   - paymentMethod, orderType (داخلي/توصيل)
   - customerId, userId, isRefunded

7. **sale_items** - عناصر المبيعات
   - id, saleId, productId, quantity, price
   - costPrice, productName, size, addonsStr

8. **shifts** - الورديات
   - id, userId, startTime, endTime
   - startCash, endCash, salesTotal, refundsTotal

9. **printers** - الطابعات
   - id, name, ipAddress, port, isReceipt

10. **customers** - العملاء
    - id, name, phone, email, notes

### إصدار قاعدة البيانات
- **الإصدار الحالي**: 13
- **نظام الترقية**: يدعم الترقية التلقائية من الإصدارات القديمة

## 🎯 الوظائف الرئيسية

### 1. نظام المصادقة والصلاحيات
- تسجيل الدخول بـ PIN
- أدوار المستخدمين (admin, cashier)
- إدارة المستخدمين والموظفين

### 2. إدارة المنتجات
- إضافة/تعديل/حذف المنتجات
- دعم المقاسات (صغير/متوسط/كبير)
- إضافة صور للمنتجات
- إدارة المخزون
- إضافة إضافات (Addons) للمنتجات

### 3. إدارة الفئات
- إنشاء فئات للمنتجات
- ربط الفئات بطابعات محددة

### 4. نظام نقاط البيع (POS)
- واجهة بيع سهلة الاستخدام
- سلة التسوق
- دعم أنواع الطلبات (داخلي/توصيل)
- اختيار العملاء
- طرق الدفع المتعددة

### 5. نظام الورديات
- فتح وإغلاق الورديات
- تتبع النقد الابتدائي والنهائي
- حساب إجمالي المبيعات والمرتجعات

### 6. العروض والخصومات
- خصومات على المنتجات
- خصومات على الفئات
- خصومات عامة
- خصومات بنسبة مئوية أو مبلغ ثابت
- تحديد تواريخ للعروض

### 7. التقارير والإحصائيات
- **ملخص عام**: إجمالي المبيعات، الأرباح، عدد الطلبات
- **تقارير المبيعات**: حسب التاريخ، الموظف، المنتج
- **تقارير المخزون**: حالة المخزون، المنتجات منخفضة المخزون
- **تقارير العملاء**: بيانات العملاء، تاريخ الشراء
- **تقارير الموظفين**: أداء الموظفين، الورديات
- **الرسوم البيانية**: رسوم بيانية تفاعلية

### 8. إدارة الطابعات
- إضافة طابعات شبكية (Network Printers)
- ربط الفئات بالطابعات
- دعم طباعة الفواتير
- دعم طباعة الطلبات حسب الفئة
- دعم اللغة العربية في الطباعة (CP1256 encoding)

### 9. إدارة العملاء
- إضافة/تعديل/حذف العملاء
- تتبع تاريخ الشراء

### 10. الإعدادات
- إعدادات المطعم (الاسم، العنوان، الهاتف)
- إدارة الموظفين
- أدوات إضافية
